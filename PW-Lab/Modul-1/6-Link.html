Python

Python adalah bahasa pemrograman tingkat tinggi yang mudah dipelajari dan digunakan. Cocok untuk pengembangan web, analisis data, kecerdasan buatan, otomatisasi, dan banyak lagi. Sintaksnya sederhana dan mudah dibaca.

JavaScript

JavaScript digunakan terutama untuk pengembangan web interaktif di sisi klien (front-end), tetapi juga dapat digunakan di sisi server (back-end) dengan Node.js. Merupakan inti dari pengembangan website modern.

Java

Java adalah bahasa pemrograman berorientasi objek yang kuat, banyak digunakan di aplikasi Android, enterprise (perusahaan), dan sistem skala besar. Tulis sekali, jalankan di mana saja (Write Once, Run Anywhere).

C#

C# (dibaca "C sharp") adalah bahasa buatan Microsoft yang digunakan untuk membuat aplikasi berbasis Windows, game dengan Unity, dan juga aplikasi web. Terintegrasi erat dengan .NET framework.

C

C adalah bahasa pemrograman prosedural yang efisien dan cepat. Digunakan untuk membangun sistem operasi, driver perangkat keras, dan aplikasi sistem lainnya. Menjadi dasar dari banyak bahasa lain.

C++

C++ adalah pengembangan dari bahasa C dengan dukungan OOP (Object-Oriented Programming). Digunakan dalam pengembangan perangkat lunak performa tinggi, seperti game, aplikasi desktop, dan mesin grafis.

Go

Go adalah bahasa yang dikembangkan oleh Google. Cocok untuk sistem terdistribusi, microservices, dan aplikasi web. Sederhana, cepat, dan efisien dalam hal performa dan pengelolaan paralelisme.

R

R banyak digunakan untuk analisis statistik, visualisasi data, dan ilmu data. Dikenal di kalangan peneliti, ilmuwan data, dan statistikawan. Mendukung berbagai paket statistik.

Swift

Swift adalah bahasa resmi untuk mengembangkan aplikasi iOS dan macOS buatan Apple. Memiliki sintaks modern, aman, dan cepat. Didesain untuk menggantikan Objective-C.

PHP

PHP adalah bahasa skrip server-side yang digunakan luas untuk pengembangan web. Banyak CMS populer seperti WordPress dibangun dengan PHP. Mudah diintegrasikan dengan HTML dan database.
