Belajar HTML

HTML adalah singkatan dari HyperText Markup Language, yaitu bahasa standar untuk membuat halaman web.

HTML digunakan untuk menyusun struktur konten di web, seperti teks, gambar, dan video.

Tag-tag HTML menentukan bagaimana elemen ditampilkan oleh browser.

Dengan menguasai HTML, kita dapat membuat halaman web yang menarik dan terstruktur.
